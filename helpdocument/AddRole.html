Role Management

Add Role

The Administrator can create new roles in addition to the roles mentioned below.

Procedure:

1.Click “Add Role”, the “Add Role” page is displayed.

Role ID Role Name Role Description
1 Administrator Administrator for the application.
2 PortOfficer PortOfficer for the application.
3 Leasee Leasee for the plot.
4 PublicUser Public User for the application.

Note: These roles cannot be deleted even by the Administrator.

2.Enter “Role Name” and “Role Description”.

3.Check the check box(s) of “Assets”.

4.Click “Add”, then New role is created.

Role Name Role Name can contain alphabets and numbers and its length should be between 3 and 20.
Role Description Role Description can contain alphabets, numbers and special characters (dot(.) and space( )) and its length should be between 5 and 50.
