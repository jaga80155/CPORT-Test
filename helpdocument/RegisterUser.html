User Management

Register User

The Administrator can register new user.

Procedure:

1.Click “Register User”, the “Register User” page is displayed.

2.Enter user credentials with which the user is to be created, personal details and select the role and click “Register”. The new user is registered.


3.Below are the criteria to enter user details:

User Name User Name should contain alphabets, numbers and special characters and its length should be between 5 and 8
Password Password should contain alphabets, numbers and special characters and its length should be between 10 and 40.Password should not be same as Username. Password should not be qwerty type.
Confirm Password Should be same as the password entered
First Name First Name can contain alphabets, numbers and its length should be between 3 and 30
Last Name Last Name can contain alphabets, numbers and its length should be between 3 and 30
Email The email’s domain name must start with alphabets, numbers and special characters i.e ("_","-",".") only, followed by @ and where second level must start with a dot “.” and length should be between 2 and 4
Phone Number Phone Number contains only numbers and its length should be between 10 and 12
