User Management

This feature enables the Administrator to create, update or delete users of CHENNAI PORT.

Role Management

This feature enables the Administrator to create new roles or restrict the role of a user. The various roles are:

ROLE DESCRIPTION
Administrator Administrator for the application
PortOfficer PortOfficer for the application
Leasee Leasee for the plot
PublicUser Public User for the application

Data Management

Through this datamanagement Page,we can publish layer into the geoserver.

Map Screen

This feature enables the user to view and operate settings, Print Screen, Plot Summary, Plot Registration, Plot Allocation report on the main screen. Users can also view all plots, available plots and booked plots.The Layers are viewed on right side of the screen.

Layers contains the base layer (OSM map) and the overlays layers.

Using Plot Registration option, users can register the plots.

Change Password

This feature allows the user to change his/her password.

Sign Out

This feature allows user to log out of the Chennai Port application.
