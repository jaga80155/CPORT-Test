Getting Started

Description:

The Port Of Chennai application can be accessed by two types of users. Based on the user role, the application allows the user to access features of Chennai Port.

To access the Web Application:

1.Enter URL in any standard web browser (Microsoft Internet Explorer (version 11 and above) /Google Chrome /FireFox Mozilla). The Home Page of Chennai Port Trust is displayed.

Note: The URL format can be of, http://(ip address of the system): (port number)/Project name

2.Provide valid Username, Password, Captcha and click on “Sign in”.

3.On successful validation, based on the user role the menu page is displayed.
